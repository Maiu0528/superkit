# What is this?

Get perfect super every time for the non-designer.

# Instalation
npm i superkit --save

Then...

'''

import { superkit } from 'superkit';

    superkit({
        super_type: 'soft',
        padding: false
});
'''

## Options
superkit supports 2 options, oth of which are optional:

* *super_type* - hard | soft (Default is soft)
* *padding* - boolean (Default is false)
